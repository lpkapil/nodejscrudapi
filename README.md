# NodeJs | Express | MongoDb - CURD API

NodeJs, Express and MognoDB CRUD API

1. Clone the repo  
```
git clone git@github.com:lpkapil/nodejscrudapi.git
```
2. Install dependencies using
```
yarn install
```
3. Srart nodejs server using
```
nodemon server.js
```
